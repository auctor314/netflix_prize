The goal of this project is to revisit the Netflix Prize problem, solve it with modern tools, and surpass the previous best result achieved by the Grand Prize winners back in 2009.

## Data

The original Netflix Prize dataset can be downloaded from Kaggle:
https://www.kaggle.com/datasets/netflix-inc/netflix-prize-data

Extract the archive into `data/raw/`, so the layout becomes:

```
data/raw/
  combined_data_1.txt
  combined_data_2.txt
  combined_data_3.txt
  combined_data_4.txt
  movie_titles.csv
  probe.txt
  qualifying.txt
  README
```
